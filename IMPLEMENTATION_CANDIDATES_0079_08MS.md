# 機動戦士ガンダム：追加候補

## 高優先：こちらだけで実装しやすい

### タムラ

- 種別: キャラクター/支援要員
- 想定ロール: operator / mechanic / support
- カード方向:
  - ホワイトベースの補給・糧食担当として、補給/回復/長期戦向けの支援キャラ。
  - 戦闘能力は低いが、母艦の補給効率やEN/弾薬回復に小さな補正を持たせると差別化しやすい。
- 台詞作成: 可能。食料・補給・現場の苦労を中心に、軽いコミカルさを出せる。

### ハロ

- 種別: キャラクター扱い、またはオプション扱い
- 想定ロール: support / mascot
- カード方向:
  - 直接戦闘ではなく、アムロやホワイトベース隊の命中/回避/士気を少し支える特殊支援。
  - OPカードにする場合は「隣接する少年兵/ニュータイプ系の回避+」のような軽量支援が合う。
- 台詞作成: 可能。ただし機械音声なので短句中心。

### カツ・レツ・キッカ

- 種別: まとめキャラクター、またはオプション
- 想定ロール: support
- カード方向:
  - 3人を個別にするより「避難民の子供たち」「カツ・レツ・キッカ」の1枚がよさそう。
  - ホワイトベース隊の士気支援、または被撃墜時の救済/根性系の小効果。
- 台詞作成: 可能。個別口調の厳密再現より、子供たちの賑やかな掛け合いとして作るのが安全。

### ゴップ

- 種別: キャラクター/連邦高官
- 想定ロール: commander
- カード方向:
  - 高い政治力・補給力、低い前線指揮という癖のある指揮官。
  - 味方の出撃コストや報酬面に補正を与えるが、前線ユニットには命中/回避の軽いマイナスを付けるなど、面白いデメリット持ちにできる。
- 台詞作成: 可能。官僚的・保身的・大局優先の口調で作れる。

### ミハル・ラトキエ

- 種別: キャラクター/スパイ・民間人
- 想定ロール: operator / support
- カード方向:
  - 情報収集・索敵・敵配置看破系。
  - カイとの相性ボーナスを付けると非常にカードらしい。
  - 戦闘力は低いが、短期的に命中/索敵を伸ばす支援役。
- 台詞作成: 可能。悲劇性を強くしすぎず、生活感と覚悟を混ぜる。

### ウラガン

- 種別: キャラクター/ジオン副官
- 想定ロール: commander / operator
- カード方向:
  - マ・クベの副官として、ギャン/マ・クベ機/鉱山基地系の支援。
  - 直接戦闘より、撤退支援・補給・罠の管理などが合う。
- 台詞作成: 可能。上官に振り回される副官味を出せる。

### デギン・ソド・ザビ

- 種別: キャラクター/ジオン総帥
- 想定ロール: commander
- カード方向:
  - 政治・講和・戦意低下/抑制系の特殊指揮官。
  - ギレンと対になる「強力だが攻撃的ではない」総帥カードにできる。
- 台詞作成: 可能。老いた権力者、戦争終結を見据える雰囲気で作れる。

### アサクラ

- 種別: キャラクター/技術士官
- 想定ロール: mechanic / commander
- カード方向:
  - ソーラ・レイ関連の技術・大型兵器支援。
  - 通常戦闘よりも、長射程/戦略兵器/固定砲台系の将来実装と相性がよい。
- 台詞作成: 可能。ただしカード効果は既存ルールに合わせて控えめにする必要あり。

## 中優先：実装可能だが役割調整が必要

### イセリナ・エッシェンバッハ

- 種別: キャラクター/民間人
- 想定ロール: support
- カード方向:
  - ガルマ関連の士気・復讐・悲劇イベント風の支援。
  - 単独カードより、ガルマ編成時の相性カード向き。
- 台詞作成: 可能。ただし戦闘台詞はかなり創作寄りになる。

### カムラン・ブルーム

- 種別: キャラクター/サイド6関係者
- 想定ロール: support / commander
- カード方向:
  - 中立コロニー・補給・外交支援。
  - ミライとの相性を少し持たせるとキャラ性が立つ。
- 台詞作成: 可能。

### エルラン

- 種別: キャラクター/連邦高官
- 想定ロール: commander
- カード方向:
  - 裏切り・情報漏洩・リスク付き指揮官。
  - 味方コストや補給にメリット、代わりに敵側へ何らかの情報優位を与えるような癖ありカード。
- 台詞作成: 可能だが、ゲーム効果を作り込むなら新スキルが必要。

### ジュダック

- 種別: キャラクター/スパイ
- 想定ロール: operator / support
- カード方向:
  - 敵情報の取得、または敵味方双方に影響するスパイ系。
  - ミハルと似るので、差別化するなら「ジオン寄りの情報員」にする。
- 台詞作成: 可能。

### マリガン

- 種別: キャラクター/キシリア側副官
- 想定ロール: commander / operator
- カード方向:
  - キシリア、ニュータイプ部隊、ア・バオア・クー終盤の補佐役。
  - 単独性能は控えめ、相性ボーナス中心がよさそう。
- 台詞作成: 可能。

### ゼナ・ザビ / ミネバ・ラオ・ザビ

- 種別: キャラクター/非戦闘
- 想定ロール: support
- カード方向:
  - ドズルやジオン側の士気支援。
  - ミネバは後年作品の比重が大きいので、初代時点では低コスト支援に留めるのが自然。
- 台詞作成: ゼナは可能。ミネバは初代時点では台詞・人格カード化は控えめが安全。

## 低優先/まとめ実装向き

- サンマロ、バンマス、マセキなどホワイトベース脇役
  - 個別カードより「ホワイトベース整備班」「避難民」などのまとめ支援がよさそう。
- アントニオ・カラス、コーリン、セキなど連邦脇役
  - 実装はできるが、既存キャラとの差別化優先度は低め。
- コム、ジェイキュー、ステッチ、ゼイガン、ソル、タチ、トワニング、バタシャム、バムロ、バロム、ビービなどジオン脇役
  - 敵ステージ用の汎用士官/兵士としては使えるが、カード化の旨味は薄い。
- ペルガミノ、ペルシア、ロラン・チュアンなど民間人
  - イベント・ステージ演出向き。

# 機動戦士ガンダム：メカ/艦艇追加候補

## 高優先

### ドダイYS

- 種別: ジオン航空支援/サブフライト
- カード方向:
  - 地上ジオンMSの移動支援、空輸、地形無視寄りの支援機。
  - グフやザク系と相性を持たせやすい。
- 実装注意:
  - 現行ルールで「輸送」表現が難しければ、低コスト飛行ユニットとして実装し、スキルで周囲の地上MS移動+1などにする。

### ドロス級空母

- 種別: ジオン宇宙母艦
- カード方向:
  - 高コスト・高耐久・低機動の空母。
  - MS補給力が高い最終盤ジオン艦。
- 実装注意:
  - グワジンとの差別化として、指揮より補給/搭載支援に寄せる。

### パゾク級輸送艦

- 種別: ジオン輸送艦
- カード方向:
  - 戦闘力は低いが補給効率がよい低〜中コスト艦。
  - パプアと被るため、宇宙専用の補給/撤退支援に寄せる。

## 中優先

### ドラゴン・フライ

- 種別: 連邦小型航空機/ヘリ
- カード方向:
  - 低コスト偵察/支援機。
  - 08小隊の戦闘ヘリ候補と似るため、まとめて航空支援枠を整理するとよい。

### ジュアッグ / アッグガイ / ゾゴック / アッグ

- 種別: ジオン水陸両用MS/特殊MS
- カード方向:
  - かなり個性的なのでカード化は楽しい。
  - ただし初代本編というよりMSV/設定寄りの印象も強いため、初代アニメ枠に入れるかは要判断。
- 実装注意:
  - 水中適性・格闘/ミサイル/特殊武装で差別化可能。

# 第08MS小隊：追加候補

## 高優先：こちらだけで実装しやすい

### アリス・ミラー

- 種別: キャラクター/連邦情報部
- 想定ロール: operator / commander
- カード方向:
  - 尋問・査問・情報操作系。
  - 敵の回避/士気を下げる、または特定条件で味方にもデメリットを与える「冷徹な調査官」カードが合う。
- 台詞作成: 可能。理性的・圧迫的・ジオン不信の強い口調で作れる。

### イーサン・ライヤー

- 種別: キャラクター/連邦指揮官
- 想定ロール: captain / commander
- カード方向:
  - 高指揮・高コストの非情な作戦指揮官。
  - 味方の攻撃力や作戦効率を上げるが、損耗時のデメリットや撤退軽視の癖を持たせると強い個性が出る。
- 台詞作成: 可能。慇懃だが冷酷、功を焦る感じ。

### キキ・ロジータ

- 種別: キャラクター/現地ゲリラ
- 想定ロール: support / commander
- カード方向:
  - 森林・山岳・ゲリラ戦術支援。
  - シローやミケルとの相性を付けられる。
  - 既存のゲリラ戦術/特殊地形ルールと相性が良い。
- 台詞作成: 可能。勝気さ、現地の地理への強さ、少女らしさを混ぜられる。

### ジダン・ニッカード

- 種別: キャラクター/連邦側の食えない補給・情報屋
- 想定ロール: support / mechanic / operator
- カード方向:
  - 物資調達、文書偽造、人脈による補給/報酬支援。
  - 直接戦闘は低いが、入手報酬や初期補給を支える変則カード。
- 台詞作成: 可能。酒・ギャンブル・人脈・軽妙な老人味。

### ジョニー・ナカミゾ

- 種別: キャラクター/連邦整備兵
- 想定ロール: mechanic
- カード方向:
  - 陸戦型ガンダム/陸戦型ジムの整備支援。
  - シローとの同郷・幼馴染要素を相性で表現できる。
- 台詞作成: 可能。整備現場寄りの台詞が作れる。

### ニエーバ

- 種別: キャラクター/ジオン整備兵
- 想定ロール: mechanic
- カード方向:
  - ノリス/グフ・カスタム専用支援。
  - グフ・カスタムのEN/弾薬/装甲を少し伸ばす、または初回被弾軽減など。
- 台詞作成: 可能。若いが腕の立つ整備兵、本国帰還への願いを混ぜる。

## 中優先

### マサド

- 種別: キャラクター/ジオン士官、ノリス副官
- 想定ロール: commander / pilot
- カード方向:
  - トーチカ防衛、ヘリ指揮、防御戦向き。
  - 戦闘ヘリコプターを実装するなら相性候補。
- 台詞作成: 可能。

### バリー / ルネン

- 種別: キャラクター/ジオン残存部隊
- 想定ロール: pilot / commander
- カード方向:
  - マゼラアタック搭乗、撤退戦・殿軍支援。
  - ボーン・アブストと3人セット相性を作ると面白い。
- 台詞作成: 可能。ただし単独の濃さはボーンより弱め。

### バレスト・ロジータ

- 種別: キャラクター/ゲリラ頭目
- 想定ロール: commander / support
- カード方向:
  - キキと並ぶゲリラ支援。
  - 森林/村/特殊地形の防衛効果。
- 台詞作成: 可能。

### ネスカ・コールマン

- 種別: キャラクター/ジャーナリスト
- 想定ロール: support / operator
- カード方向:
  - 直接戦闘より、報酬・士気・戦争被害テーマの特殊支援。
  - ステージクリア報酬や敵撃破以外の評価軸を作るなら面白い。
- 台詞作成: 可能だが、サウンドトラックCD由来なので採用範囲は要確認。

## 低優先/まとめ実装向き

### チビ / ノッポ / ヒゲ

- 種別: 民間ゲリラ
- カード方向:
  - 個別より「現地ゲリラ」または「ロジータ村ゲリラ隊」でまとめた方がカードとして扱いやすい。
  - キキ/バレストの支援対象にするのが自然。

## 第08MS小隊：メカ/兵器追加候補

### MS-06RD-4 宇宙用高機動試験型ザク

- 種別: ジオンMS
- 優先度: 高
- カード方向:
  - アイナの序盤搭乗機として実装価値が高い。
  - 宇宙専用、高機動、脆め、武装少なめ。既存の高機動型ザクとは「試験機」「アイナ適性」で差別化。
- 制約:
  - ザク系なので、明確な理由がなければ携行ビーム不可。

### RGM-79E 初期型ジム

- 種別: 連邦MS
- 優先度: 高
- カード方向:
  - 第1話宇宙戦の連邦側MS。
  - ジムより少し試験機寄り/宇宙寄りの低〜中コスト機体として作れる。

### MS-05B トップ専用ザクI

- 種別: ジオンMS
- 優先度: 中〜高
- カード方向:
  - トップは既に実装済みなので専用機として実装しやすい。
  - 性能は低めだが、トップ搭乗時に生存/撤退/村イベント風の補正を持たせる。
- 制約:
  - ザクI系なので携行ビーム不可。

### MS-06J アス機 / デル機

- 種別: ジオンMS
- 優先度: 中
- カード方向:
  - 汎用ザクIIとの差別化は薄い。
  - もし入れるなら、トップ小隊セット運用や村イベント再現として実装。
- 制約:
  - ザクII系なので携行ビーム不可。

### ドダイII

- 種別: ジオン航空支援
- 優先度: 中〜高
- カード方向:
  - 08小隊版の地上航空支援。
  - ドダイYSと同時に整理して、ジオン地上MSの移動支援枠にするのがよさそう。

### 戦闘ヘリコプター

- 種別: ジオン低コスト航空ユニット
- 優先度: 中
- カード方向:
  - マサドとセットで防衛戦の低コスト航空戦力。
  - 索敵・低火力・高移動・脆い、という役割。

### トーチカ/固定砲台系

- 種別: 固定兵器
- 優先度: 低〜中
- カード方向:
  - 通常カードよりステージギミック向き。
  - 防衛マップを作るなら実装価値あり。
