# Gundam-style SRPG Prototype

ブラウザで遊ぶ、カード収集型SRPGプロトタイプです。

機体、武装、キャラクター、戦艦を組み合わせて部隊を編成し、地形と射程を見ながら敵部隊を撃破します。

## 起動方法

`index.html` をブラウザで開くとプレイできます。

ローカルサーバーで確認する場合は、リポジトリ直下で次のように起動します。

```bash
python -m http.server 8765
```

その後、ブラウザで `http://localhost:8765/` を開いてください。

## 基本の流れ

1. ステージを選びます。
2. 勢力、戦艦、艦長、副長、MS、武装、オプション、キャラクターを選んで部隊を編成します。
3. コスト上限内に収めて出撃します。
4. 配置フェイズでは、自軍ユニットを明るい配置可能マスへ置きます。
5. 自軍ユニットを選び、緑のマスへ移動します。
6. 敵ユニットをクリックして攻撃対象にし、武装ボタンで攻撃します。
7. 行動が終わったらターン終了を押します。
8. 敵を全滅させると勝利、自軍戦艦が撃沈されると敗北です。

## フリー対戦

- フリー対戦では、所持カード内でコスト上限なしに編成できます。
- 各マップは、対応するステージを初回クリアするとフリー対戦で解放されます。
- 敵は反対勢力の全カードから、戦艦1隻を含む自軍総コストに近いランダム編成で出撃します。
- シチュエーション固定の敵編成は使いません。
- 勝利すると全体ランダムドロップからカードを1枚入手できます。
- フリー対戦の勝敗は、ステージクリア状況やカード引換券には影響しません。

## 編成について

- 「おすすめ編成」を押すと、現在の勢力・ステージ・所持カード・地形・コスト上限に合わせて、戦艦からMS、乗員、武器、オプションまで自動で編成します。作成後は手動で調整できます。
- 編成画面では、前回使った編成が勢力ごとに記憶されます。
- よく使う編成は「お気に入り編成」に20枠まで登録できます。
- 同じ人物は、別カード扱いでも同時に編成できません。
- キャラクターはMSにも戦艦にも配置できますが、配置先によって有効な能力やスキルが変わります。
- 戦艦には艦長と副長を配置できます。

## キャラクター能力の見方

- MS搭乗時は、主に射撃・格闘・反応・覚醒と、MS向けスキルが機能します。
- 戦艦の艦長時は、主に指揮が機能します。
- 戦艦の副長時は、主に通信・整備が機能します。
- 得意役割と違う場所にも配置できますが、一部の能力やスキルが働かないことがあります。
- 迷ったら、カード詳細の効果説明を確認してください。

## キャラクター成長

- 勝敗が決まるところまで出撃したキャラクターは、20回出撃するごとに成長ポイントを1獲得します。
- 成長ポイントは、キャラクター詳細から射撃・格闘・反応・指揮・通信・整備のいずれかに使用できます。
- 各能力の上限は30です。成長内容はそのセーブデータで自分が使う時だけ反映されます。
- 出撃準備から戻っただけでは出撃回数に加算されません。

## 戦闘の主なルール

- 武器ごとに1回ずつ攻撃できます。
- MSは1ターン中に3回目以降の攻撃を行うと、命中率と最低命中率が段階的に低下します。
- 戦艦には連続攻撃ペナルティがありません。
- 武器には射程、最低射程、EN消費、弾数、障害物越し攻撃の可否があります。
- チャージ式武器は、移動・攻撃前に必要回数の「チャージ」を行うと発射できます。チャージしたターンの行動は終了します。
- ビーム、実弾、格闘、盾など、武器種別によって影響するスキルや防御効果が異なります。
- 戦艦に隣接してターンを終えると、EN、実弾、装甲、盾が補給・修理されます。
- 一部の武器やスキルは、ステルス、偵察、機雷、Iフィールド、チームワークなどの特殊効果を持ちます。
- バリケード設置は1戦闘に1回、未移動時の行動として前方1マスへ破壊可能な障害物を作ります。バリケードは移動と通常射撃の射線を敵味方ともに遮ります。
- 優先目標指示は未移動時、指揮に応じた範囲内の可視敵1機を指定します。使用者は移動不能になりますが攻撃でき、そのターン中は味方全体が指定対象への命中+8を得ます。
- 精密攻撃管制は1ターンの攻撃回数を1回に制限する代わりに、その攻撃の命中+12、与ダメージ+10を得ます。
- 緊急修理は1戦闘に1回、行動として自機の装甲を「30＋整備×2」回復します。回復上限は最大装甲の20%で、EN・弾薬・盾は回復しません。
- トレイル・フォーメーションは同じ縦列かつ4マス以内に別の生存味方MSがいる間、条件内の自機と味方MSへ回避+5、被ダメージ-6を与えます。
- ライン・フォーメーションは同じ横列かつ4マス以内に別の生存味方MSがいる間、条件内の自機と味方MSへ命中+5、与ダメージ+6を与えます。
- アイナの懐中時計は通常ダメージで装甲0になった時、一度だけ装甲1で踏みとどまります。脱出機能・増加装甲持ちには装備できず、強制撤退や自爆は防げません。
- ラストシューティングは直接攻撃で撃破された時、一度だけ撃破した敵へ射程内かつ使用可能な最適武装で即座に反撃します。命中判定とEN・弾薬消費は通常どおりです。
- テム・レイのパーツは編成コストを30下げる代わりに、最大装甲・最大EN・運動・移動力・命中・与ダメージを大幅に低下させるネタオプションです。
- 隠密中の敵は機体・キャラ・装備詳細を確認できません。偵察機の索敵範囲や接近など、効果ごとの看破条件を満たすと確認できます。
- 偵察機が索敵範囲内の隠密を看破すると、味方全体が対象の詳細確認と射撃を行えます。索敵範囲は搭乗者の通信が高いほど2～5マスへ拡張されます。
- 煙幕やアクティブ・カモなど、戦闘中に発動する隠密は、それ以前の看破状態を上書きできます。敵偵察圏内で発動してもそのターン中は隠密になりますが、次の自軍ターン開始時にも偵察圏内なら解除されます。
- 看破による永続的な隠密無効はありません。隠密再発動と再看破で状態が上書きされます。

## 地形について

- マップには地上、宇宙、コロニー、空中があります。
- 空中マップへ出撃できるのは飛行ユニットか、飛行対応SFSを装備した地上用MSです。
- 水中、森林、砂漠、デブリ帯では、適性のない機体の移動力と回避率が下がります。
- 飛行ユニットは、地上の特殊地形ペナルティを無視します。
- 障害物マスは移動できず、通常武器の射線を遮ります。
- クラッカーや一部砲撃武器は、障害物を越えて攻撃できます。

## カード収集と報酬

- 勝利すると全体ランダムドロップや引換券でカードを入手できます。
- ランダムドロップはカード種別ごとの重みで抽選され、未所持や所持数の少ないカードが少し出やすくなります。
- 入手したカードはコレクションに保存され、以後の編成で使えます。
- セーブデータはブラウザのローカルストレージに保存されます。
- 「進行を初期化」を使うと、所持カード、前回編成、お気に入り編成、キャラクター成長が初期化されます。
